# Sales-Dashborad-2023---2026-
📊 Sales Dashboard (2023 – 2026)

This project presents an interactive Sales Dashboard built in Power BI to analyze computer hardware product sales across different brands, categories, and states in India. The dashboard provides a clear overview of sales performance, cost, profit, and order quantity to support business decision-making.

🎯 Project Objective

To visualize and analyze sales data from 2023 to 2026.

To track sales performance by brand, product category, and location.

To provide quick insights into total sales, total cost, profit, and order quantity.
